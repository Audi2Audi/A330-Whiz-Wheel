# A330 Whiz Wheel

A quick-reference calculator for the A330 rule of thumb:

**595 − Gross Weight (in 1000kg) = Max Flight Level**

e.g. at 225.0t gross weight: 595 − 225 = 370 → max FL370.

## What it does

- Shows your current max altitude on a dial styled after a classic E6-B flight computer, with a needle against a flight-level scale.
- Given your current gross weight, fuel burn rate, and the time right now, calculates the estimated UTC time you'll be light enough to accept a specific flight level ATC requests (e.g. "when can you accept FL390?").
- Shows a quick-reference table across FL290–FL410 all at once, so you can see several options without re-entering numbers.

## Use

Open `index.html` directly, or host it (e.g. GitHub Pages) and add it to your phone's home screen for one-tap access.

This is a rule-of-thumb estimate only — always confirm against FCOM/FCTM performance data and current OFP figures before accepting an ATC altitude assignment.
